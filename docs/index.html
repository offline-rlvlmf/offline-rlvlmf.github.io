Combining RL-VLM-F with offline RL for policy learning from unlabeled datasets

Abstract

Offline reinforcement learning can enable policy learning from pre-collected, sub-optimal datasets without online interactions. We propose a novel system that uses preference feedback from vision-language models and textual task descriptions to generate reward labels automatically for offline datasets. This system is tested on complex real-world robot-assisted dressing tasks and shows significant improvement over baseline methods.
